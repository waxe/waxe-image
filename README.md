# Requirements
- Python 2.7
- virtualenv


# Installation
```
mkdir mywebsite
cd mywebsite
wget https://raw.githubusercontent.com/waxe/waxe-image/develop/install.sh
sh ./install.sh
wget https://raw.githubusercontent.com/waxe/waxe-image/develop/production.ini
initialize_waxe-image_db production.ini
```
